# agente_viajero
Este proyecto implementa una solución al Problema del Agente Viajero utilizando un algoritmo genético.
El objetivo del algoritmo es encontrar la ruta más corta posible que visite un conjunto de municipios exactamente una vez y regrese al punto de origen.
INSTRUCCIONES PARA CORRER EL CODIGO:
Asegúrate de tener Python instalado.
si quieres verificar que lo tengas instalado, puedes verificarlo con:

python --version

Instala las dependencias necesarias:

pip install numpy

Ejecuta el programa principal:

python viajero.py

Esto iniciará el algoritmo genético y mostrará en consola la distancia inicial y la mejor ruta encontrada tras las generaciones.
